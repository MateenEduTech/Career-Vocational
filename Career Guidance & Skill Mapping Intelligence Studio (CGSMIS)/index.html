📱 APK Conversion Guide

Option A — PWA Install (Fastest)

Open in Chrome on Android → tap ⋮ → "Add to Home Screen". Installs as a native-like app with offline support.

Option B — Android Studio WebView

New Project → Empty Activity → Replace layout with WebView → Load file:///android_asset/index.html → Enable JavaScript + DOM Storage → Copy all files to assets/ → Build Signed APK

Option C — PWABuilder.com

Host on GitHub Pages/Netlify → Visit PWABuilder.com → Enter URL → Download Android package.
